# Space In V2

## 主要功能
此為提供補習班空間預約功能,主要功能概述為：
1. [空間預約] 可選擇補習班預約教室(包含基本時間卡控)。
2. [已預約] 檢視預約者預約狀況，並取消預約。
3. [空間總覽] 空間擁有者可檢視該空間預約情況。
4. [購買方案] 串接第三方綠介金流，供補習班付款。
